# slack public channel

Slack の public チャンネルを Google Spread に書き出すスクリプト

# Getting Started

## Installing clasp

https://github.com/google/clasp

## Deployment

```
npm install
clasp push
```

## Parameter Configuration

### Script Property

名前 | 説明
---|---
`SLACK_BOT_TOKEN` | Slack App のトークン

## Running Script

GAS 上で実行してください
